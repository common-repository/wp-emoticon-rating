== Description ==

WP Emoticon Rating is wordress plugin which allows your visitors to rate your posts and pages.,
visitors can express their rating in the form of emoticons, very easy to use,Just install and activate the plugin and set global the settings to use,
It is very eye catching display of emoticons bellow your posts and pages.

== Installation ==
1. Download the plugins
2. upload and Extract files to /wp-content/plugins/ directory
3. Go to backend and activate plugin and you are ready to go.
4. Go to setting > Emoticon Rating.
